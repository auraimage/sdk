# Security Policy

## Reporting a Vulnerability

If you discover a security issue in `@auraimage/sdk`, please report it privately to **security@auraimage.ai** rather than opening a public GitHub issue.

Include:

- A description of the vulnerability and its impact.
- Steps to reproduce, including the SDK version affected.
- Any proof-of-concept code, if available.

We aim to acknowledge reports within 2 business days and to ship a fix on the earliest reasonable release.

## Supported Versions

The latest `0.x` release is supported. Once a `1.0` is published, support will extend to the two most recent majors.

## Scope

This policy covers the `@auraimage/sdk` npm package. Vulnerabilities in the Auraimage hosted service (auraimage.ai, cdn.auraimage.ai, etc.) should also be reported to security@auraimage.ai.
